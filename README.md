# C8_Project1_BankExtension2

A comprehensive banking system developed in C++ that manages clients and users with a professional permission-based access control. The system uses flat-file storage to ensure data persistence across sessions.

### 🚀 Key Features:

* **Client Management:** Complete CRUD operations (Create, Read, Update, Delete) for client profiles.
* **User Management:** Admin panel to manage system users and assign specific permissions.
* **Transaction Module:** Supports deposits, withdrawals, and total balance tracking with validation checks.
* **Secure Login System:** Multi-user login with encrypted-like permission levels using Bitwise logic.
* **Data Persistence:** Efficiently handles data reading/writing from `.txt` files.

### 🛠 Technical Skills Demonstrated:

* **Data Structures:** Using `struct` and `vector` for complex data management.
* **File I/O:** Advanced file handling and data parsing.
* **Bitwise Logic:** Implementation of a robust permission system.
